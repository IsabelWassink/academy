WikiPathways: Jenkins report

Goal

To review the results from the automated Jenkins tests. The tests check for many aspects of WikiPathways curation, and the list of failed tests corresponds to curation issues in WikiPathways.

Unknown DataNode Type

  • On the report page, expand the entries ending in DataNodes.unknownTypes.
  • Find the WPID for an affected pathway.
  • Open the relevant pathway in PathVisio using the WPID, then find the problematic node by the label listed in the Jenkins report.
  • If it's clear what type the node should be, assign it. If it's not a data node, redraw as graphical element. Pro-tip: One can use the TypeConverter plugin to quickly change the type of a Data Node.

Weird Characters in Titles

  • Update title at WP

Duplicate Titles

  • Review the affected pathways to compare them. If one is just a copy of the other with no other new information, delete it.
  • If the pathways are unique, update one of the titles to better reflect the content.
